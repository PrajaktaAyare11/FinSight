"# Personal Finance Visualization System" 
A web-based platform to help users track, visualize, and manage their personal finances with advanced analytics and personalized insights.

## Features
- Bank transaction categorization
- Spending trend charts
- Savings goal tracking
- AI-powered spending recommendations
- Unique feature: Emotional spending detection
- Unique feature: Gamified financial milestones

## Tech Stack
- Frontend: Next.js, TailwindCSS
- Backend: Node.js with Express
- Database: MySQL with Prisma ORM
- Hosting: Vercel (frontend), Render/Heroku (backend)
